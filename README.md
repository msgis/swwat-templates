# SwwAT GIS Datentemplates

Dieses repository enthält GIS Datentemplates (QGIS und ArcGIS), deren zugrundeliegende Datenstruktur den Vorgaben der jeweiligen digitalen Datenanforderungen des Bundes entsprechen und zur Übernahme in die Hochwasserfachdatenbank zur Verfügung gestellt werden.

Die GIS Datentemplates im repository selbst sind in Projektion MGI / Austrian Lambert (EPSG: 31287) abgelegt. Alle GIS Datentemplates sind weiters als Datenpakete im .ZIP-Format getrennt nach Verwendung in QGIS oder ArcGIS und jeweils gängigen Projektionen auf der Subseite swwat-templates/releases zum Download verfügbar.
